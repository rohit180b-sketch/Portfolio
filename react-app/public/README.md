Place your profile and about images here so Vite serves them at the app root.

Recommended filenames (used by the app):
- profile-pic1.jpg
- pic2.jpg

Copy the files into this folder before running `npm run dev` or `npm run build`.

Example (PowerShell):

```powershell
# from workspace root
copy "..\profile pic1.jpg" "react-app\public\profile-pic1.jpg"
copy "..\Pic2.JPG" "react-app\public\pic2.jpg"
```
